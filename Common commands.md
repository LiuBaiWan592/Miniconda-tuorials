# Conda常用命令（Conda Common Commands）

## 一、管理conda

1. 查看conda版本

   ```bash
   conda --version
   conda -V
   ```

2. 列出所有的环境

    ```bash
    conda env list
    # conda info --envs
    ```

    *`conda list`命令用于查看conda下的包，而`conda env list`命令可以用来查看conda创建的所有虚拟环境。*

3. 查看环境管理的全部命令帮助

    ```bash
    conda env -h
    ```
